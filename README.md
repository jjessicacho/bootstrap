# Bootstrap

Although Bootstrap is a bit outdated nowadays, I wanted to practice how grid works by creating a simple blog. As a result, I learned grid, carousel, and how to name CSS class and ID selectors. 

## Features
- [x] jumbotron
- [x] alert
- [x] header
- [x] navigation + menu
- [x] images
- [x] articles
- [x] table
- [x] carousel
- [x] footer
